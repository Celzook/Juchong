# 📋 주주총회 일정 트래커

## 📁 파일 구성

```
agm_tracker/
├── app.py              ← Streamlit 앱 메인 파일
├── requirements.txt    ← 패키지 목록
├── 주주총회.xlsx        ← 데이터 파일 (여기에 복사)
└── agm_state.json      ← 상태 저장 (자동 생성)
```

---

## 🚀 실행 방법

### 1. 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 엑셀 파일 배치

`주주총회.xlsx` 파일을 `app.py` 와 같은 폴더에 복사하세요.

### 3. 앱 실행

```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 로 자동 접속됩니다.

---

## 📖 기능 설명

### DART 자동 검색
- **개별 검색**: 각 기업 행의 🔍 DART 버튼 클릭
- **전체 검색**: 사이드바 > 🔍 전체 기업 DART 검색
- 날짜가 변경된 기업에는 `🔄 업데이트됨` 배지 표시 (노란 반짝임 효과)

### DART OpenAPI 키 (선택, 권장)
1. https://opendart.fss.or.kr 회원가입 및 API 키 발급 (무료)
2. 사이드바에 키 입력
3. 더 정확하고 빠른 검색 가능

### 기업 변경
1. ✏️ 기업변경 버튼 클릭
2. 새 기업명 + 주주총회 날짜 입력
3. 확정 클릭
4. 기존 기업은 **변경 1회 전 기업**으로 기록됨
5. 새 기업명(▶ 표시) 클릭 시 이전 기업 정보 확인 가능

### 상태 초기화
- **업데이트 표시 초기화**: 노란 배지만 제거 (날짜는 유지)
- **전체 상태 초기화**: 모든 변경사항 초기화

## 💡 팁

- 상태는 `agm_state.json` 에 자동 저장되므로 앱을 껐다 켜도 유지됩니다.
- DART API 없이도 웹 크롤링으로 동작하지만, API 사용 시 더 안정적입니다.
- 전체 검색 시 DART 서버 부하를 줄이기 위해 기업당 0.3초 간격으로 검색합니다.
